# Synesthesia-of-Machines-COMST
X. Cheng et al., "Intelligent Multi-Modal Sensing-Communication Integration: Synesthesia of Machines," in IEEE Communications Surveys & Tutorials, vol. 26, no. 1, pp. 258-301, Firstquarter, 2024. 

## Dependencies and Installation
- Python 3.9 (Recommend to use [Anaconda](https://www.anaconda.com/))
- Pytorch 2.0.0
- NVIDIA GPU + CUDA
